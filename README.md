# Gadget Shop System

# Description

This project is a Java-based Gadget Shop system developed using JavaFX.
It allows users to create and manage different types of gadgets, including mobile phones and MP3 players.

The system demonstrates object-oriented programming concepts such as:

* Inheritance
* Encapsulation
* Method overriding
* ArrayList usage
* GUI development using JavaFX

---

# Classes Included

* `Gadget.java` – Base class containing common attributes (model, price, weight, size)
* `Mobile.java` – Subclass of Gadget with calling credit functionality
* `MP3.java` – Subclass of Gadget with memory management functionality
* `GadgetShop.java` – GUI application that allows user interaction

---

# Features

* Add a Mobile phone
* Add an MP3 player
* Display all gadgets with index number
* Make a phone call (for Mobile)
* Download music (for MP3)
* Input validation using try/catch
* Error messages using dialog boxes
* Clear all input fields

---

# How to Run the Program

Open Command Prompt in the project folder and run:

```
javac *.java
java GadgetShop
```
